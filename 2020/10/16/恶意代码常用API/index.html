1、文件系统函数

1、CreateFile

这个函数被用来创建和打开文件。

2、ReadFile和WriteFile

这些函数被用来Madui对文件进行读和写。

C3、CreateFileMapping和MapViewOfFile

CreateFileMapping函数复制从磁盘上加载一个文件到内存中。MapViewOfFile函数返回一个指向映射的基地址指针。
